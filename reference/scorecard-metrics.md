# Scorecard Metrics

> Weekly operating scorecard. Update every Monday. Values flow into `context/current-data.md`.

---

## How to Use

1. Open this file every Monday
2. Fill in the current week's values for each metric
3. Compare against target — flag anything red
4. Update `outputs/project-register.md` if project status has changed

---

### 4. Conversion Rate

| Field | Value |
|---|---|
| **Definition** | % of qualified conversations that convert into a paid engagement |
| **Formula** | Deals closed ÷ qualified calls / demos × 100 |
| **Target** | [YOUR_TARGET] |

| **Source** | CRM / project register |
| **Owner** | Founder |
| **Notes** | Low conversion = wrong ICP reaching you, or offer not resonating |

### 5. Delivery Cycle Time

| Field | Value |
|---|---|
| **Definition** | Calendar days from project kick-off to completion / sign-off |
